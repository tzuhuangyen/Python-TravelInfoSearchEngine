安裝與使用

前提條件

  • Python 3.6+
  • Chrome 瀏覽器
  • Chrome WebDriver (與您的 Chrome 版本相匹配)

安裝步驟

# 克隆儲存庫 git clone https://github.com/tzuhuangyen/Python-TravelInfoSearchEngine.git travel-search-engine # 安裝依賴 pip install -r requirements.txt # 運行應用 python app.py

使用方法

  1. 在首頁輸入框中輸入想要搜尋的旅遊關鍵字
  2. 點擊「搜尋」按鈕或按 Enter 鍵
  3. 或者點擊預設的熱門目的地標籤
  4. 查看搜尋結果頁面上來自各平台的文章列表
  5. 點擊「閱讀完整文章」可跳轉到原始文章頁面
