旅遊文章搜尋引擎

一站式搜尋多平台旅遊文章，輕鬆規劃您的下一次旅行

查看 GitHub 代碼 了解功能

主要功能

多平台整合搜尋

同時從 Medium、DCard、PTT、背包客棧等多個平台搜尋旅遊文章，節省您的時間。

熱門目的地標籤

提供熱門旅遊目的地標籤，一鍵搜尋日本、韓國、泰國、英國、西班牙等熱門地區的旅遊文章。

響應式設計

無論在桌面電腦、平板或手機上，都能獲得最佳的使用體驗。

安裝與使用

前提條件

  • Python 3.6+
  • Chrome 瀏覽器
  • Chrome WebDriver (與您的 Chrome 版本相匹配)

安裝步驟

# 克隆倉庫 git clone https://github.com/YOUR_USERNAME/travel-search-engine.git cd travel-search-engine # 安裝依賴 pip install -r requirements.txt # 運行應用 python app.py

使用方法

  1. 在首頁輸入框中輸入想要搜尋的旅遊關鍵字
  2. 點擊「搜尋」按鈕或按 Enter 鍵
  3. 或者點擊預設的熱門目的地標籤
  4. 查看搜尋結果頁面上來自各平台的文章列表
  5. 點擊「閱讀完整文章」可跳轉到原始文章頁面
